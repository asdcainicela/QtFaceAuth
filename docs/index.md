# Bienvenido a QtFaceAuth

**Sistema de autenticación biométrica facial de alto rendimiento.**

Este sistema utiliza C++ y Qt 6 para proporcionar una interfaz de alta velocidad y procesamiento de imágenes en tiempo real.
